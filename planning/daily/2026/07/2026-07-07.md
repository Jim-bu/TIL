## 💡 Tech ↔ Product 아키텍처 디자인 매핑

### ⚙️ [패턴 1] 에이전트용 실시간 데이터 공급망 (Data Pipeline)
- **GitHub firecrawl** ↔ **PH AnySearch**: 에이전트가 발전할수록 과거가 아닌 '실시간' 데이터를 원함 → 엔지니어는 `firecrawl` 같은 고성능 크롤러를 분산 큐로 묶어 운영.
- **기술 챌린지:** IP 차단·가짜 페이지 필터링을 위한 프록시 로테이션, 비구조화 HTML을 토큰 효율 위해 1차 Markdown 정제 → 벡터 DB/JSON 가공 파이프라인.

### 🧠 [패턴 2] 프라이버시-우선 에지 AI (Edge / On-device AI)
- **GitHub meetily** ↔ **PH Typeahead 2.0**: 생산성 툴은 데이터 유출 방지(Privacy)와 Zero-latency가 생명 → 둘 다 클라우드 전송 없이 로컬 자원(NPU/GPU) 사용.
- **기술 챌린지:** `WebAssembly`/`CoreML`로 3B~8B급 SLM을 메모리 누수 없이 서빙하는 법. meetily의 Tauri+whisper.cpp+Ollama 조합이 실제 참고 구현체.

## 📚 오늘 퇴근 후 Deep Dive 1개 Pick

- **키워드:** **Model Context Protocol (MCP)**
- **이유:** `claude-skills`·`agent-skills`는 *Skill(문서)* 이지만, 이들을 실제 에이전트가 로컬 환경(IDE·파일시스템·API)과 안전하게 연결하는 표준으로 **MCP**가 함께 언급됨. → *Skill과 MCP는 별개지만 상호보완* 이라는 점이 핵심 학습 포인트.
- **방향:** Anthropic 공식 MCP 가이드 → 에이전트가 로컬 도구 호출 시 보안(Sandboxing)을 어떻게 유지하는지 아키텍처 문서 확인.
